## 一、官方文档（免费 · 最权威 · 随时查阅）

> **2025+ 重要变化**：NVIDIA 主文档已重组为 [CUDA Programming Guide](https://docs.nvidia.com/cuda/cuda-programming-guide/)（5 部分），官方最近更新（2026-05）已**完整发布 Part 3/4/5 全部小节**。  
> 旧链接 [cuda-c-programming-guide (Legacy)](https://docs.nvidia.com/cuda/cuda-c-programming-guide/) 仍可用，但 **章节编号与新版不同**。  
> 本项目学习路线均按 **新版** 编写；Legacy `Ch.x` 对照见 [Week1详细步骤.md 附录 D](Week1详细步骤.md#附录-d新版-vs-legacy-章节对照查阅用)。

**新版完整结构速查**

| Part | 包含小节（官方最新） | 学习阶段 |
|------|------|----------|
| **1. Introduction to CUDA** | 1.1 Introduction、1.2 Programming Model、1.3 The CUDA platform | Week 1 Step 02 |
| **2. Programming GPUs in CUDA** | 2.1 Intro to CUDA C++、2.2 Intro to CUDA Python、2.3 Writing SIMT Kernels、2.4 Writing Tile Kernels、2.5 Asynchronous Execution、2.6 Unified and System Memory、2.7 NVCC | Week 1–5 |
| **3. Advanced CUDA** | 3.1 Advanced CUDA APIs、3.2 Advanced Kernel Programming、3.3 Driver API、3.4 Multiple GPUs、3.5 Tour of CUDA Features | Week 3–7 |
| **4. CUDA Features** | 4.1 Unified Memory、4.2 CUDA Graphs、4.3 Stream-Ordered Allocator、4.4 Cooperative Groups、4.10 Pipelines、4.11 Async Data Copies、4.13 L2 Cache Control、4.18 Dynamic Parallelism 等共 20 节 | 按需查阅 |
| **5. Technical Appendices** | 5.1 Compute Capabilities、5.2 Env Variables、5.3 C++ Language Support、5.4 C/C++ Language Extensions、5.5 Floating-Point、5.6 Device-Callable APIs、5.7 Memory Model、5.8 Execution Model | Week 1+ 随时 |

> **常用内联函数在哪里（易踩坑）**：`__syncthreads`/`__syncwarp` → **5.4.4**；`atomicAdd` 等原子操作 → **5.4.5**；`__shfl_down_sync`/warp vote/reduce → **5.4.6**；WMMA/Tensor Core → **5.4.11**；`__ldg`/向量加载 → **5.4.8**。

## 二、书籍

| 书名 | 作者 | 适合阶段 | 建议 |

## 六、工具链

| 工具 | 用途 | 安装/使用 |
|------|------|-----------|
| `nvcc` | CUDA 编译器 | 已安装 CUDA 13.3 |
| `nvidia-smi` | 设备状态 | 系统自带 |
| `nsys` | Nsight Systems CLI | `sudo apt install nsight-systems` 或 CUDA 配套 |
| `ncu` | Nsight Compute CLI | CUDA 配套 |
| `cuda-gdb` | GPU 调试 | CUDA toolkit |
| `compute-sanitizer` | 内存/竞态检测 | `compute-sanitizer --tool memcheck ./app` |
| CMake ≥ 3.18 | 构建 | `FindCUDAToolkit` |
| Python + matplotlib | Roofline 绘图 | Week 5 |

**T4 上建议安装**

```bash
# 内存检测（每次大改 kernel 后跑一遍）
compute-sanitizer --tool memcheck ./your_app

# 性能分析
nsys profile -o report ./your_app
ncu --set full -o profile ./your_app
```
