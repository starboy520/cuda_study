## 3. 索引答案

### 一维

```text
blockDim.x=256
blockIdx.x=4
threadIdx.x=7
index=1031
```

### 二维

```text
block=(16,8)
blockIdx=(2,3)
threadIdx=(5,6)

col=37
row=30
```

行主序地址为：

```text
30 * width + 37
```

前提是 row/col 都在边界内。

## 4. 函数修饰符答案

| 修饰符 | 调用方 | 执行方 |
|---|---|---|
| 普通/`__host__` | Host | Host |
| `__device__` | Device | Device |
| `__global__` | Host launch | Device |
| `__host__ __device__` | 两边 | 分别生成版本 |

## 5. 错误时机

```text
非法 block 配置 -> launch check
kernel 越界      -> 执行/同步时暴露
```

实际 CUDA API 可能返回此前的异步错误，因此报告时要记录最早可靠检查点。

## 6. 编译链

```text
NVCC 驱动
Host code -> Host compiler
Device code -> PTX/cubin
Fat binary -> executable
Driver 选择 cubin 或 JIT PTX
```
